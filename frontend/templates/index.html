About the Models

Groq Model

Uses Llama 3.3 70B Versatile hosted on Groq's ultra-fast inference platform. Groq's custom LPU (Language Processing Unit) hardware delivers responses in 1-2 seconds.

  • 70 billion parameters
  • Cloud-based inference
  • State-of-the-art open model
  • Best for quick, accurate answers

Custom Model

A 125.8M parameter transformer I trained from scratch specifically for quantum computing Q&A. This demonstrates end-to-end ML engineering skills.

  • Custom architecture (RoPE, RMSNorm, SwiGLU)
  • Trained on 28,071 Q&A pairs
  • Two-phase training: book pretraining + Q&A fine-tuning
  • RAG system for knowledge retrieval
